# Setup Guide

## Step 1

Install VirtualBox.

## Step 2

Download Windows 10 ISO from Microsoft.

## Step 3

Create VM:

Name: Windows-Lab-10

Type: Microsoft Windows

Version: Windows 10 (64-bit)

## Step 4

Configure Hardware

RAM: 16GB

CPU: 4

Video Memory: 128MB

## Step 5

Create Virtual Disk

Format: VDI

Storage: Dynamically Allocated

Size: 64GB

## Step 6

Mount ISO

Settings → Storage → Optical Drive → Choose ISO

## Step 7

Start Installation

Complete Windows setup.

## Step 8

Install Guest Additions

Devices → Insert Guest Additions CD Image
